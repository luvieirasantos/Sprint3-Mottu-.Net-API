Bem-vindo à API Mottu

Esta é a API RESTful para gestão de funcionários, gerentes e pátios da Mottu.

Use os endpoints disponíveis para cadastrar e gerenciar os recursos.

Funcionalidades:

  • ✅ Autenticação de funcionários
  • ✅ Cadastro de funcionários com hash de senha
  • ✅ Gestão de pátios
  • ✅ Gestão de gerentes
  • ✅ Paginação em todos os endpoints
  • ✅ Links HATEOAS para navegação
